# Workflow: 内容生成

## 触发条件

用户输入以下指令时触发：
- "生成下一篇笔记"
- "写一篇关于[主题]的笔记"
- "爆款选题"
- "帮我写一篇"
- "创作新内容"

## 工作流步骤

### 步骤 1：热点获取

**Agent**: CatHotTopicAgent

**操作**:
1. 使用 agent-reach 跨平台搜索美短相关热点
2. 合并去重
3. 按热度排序取 TOP50

**输出**: 热点列表

---

### 步骤 2：历史去重

**Agent**: ContentDeduplicator (工具)

**操作**:
1. 加载历史内容库 `memory/content_history.json`
2. 对每个热点进行语义相似度检测
3. 标记重复风险
4. 过滤掉高风险重复内容

**输出**: 去重后热点列表

---

### 步骤 3：选题评分

**Agent**: TopicRanker

**操作**:
1. 综合热点热度、账号匹配度、用户兴趣、爆款概率、差异化
2. 减去重复风险分
3. 排序取 TOP10

**输出**: TOP10 推荐选题

---

### 步骤 4：用户确认（可选）

Orchestrator 展示 TOP10 选题，让用户选择或确认：

```
📋 今日推荐选题 TOP10

1. 🥇 银虎斑美短文身基因解析 (评分: 85.5)
2. 🥈 美短文身品种鉴定指南 (评分: 82.3)
3. 🥉 美短夏日掉毛急救指南 (评分: 80.1)
...

请选择（输入编号），或直接告诉我你想写什么：
```

---

### 步骤 5：生成笔记

**Agent**: XHSWriter

**操作**:
1. 读取选题和账号画像
2. 按账号风格生成笔记
3. 确保标题 ≤ 20字，正文 600-900字
4. 包含互动引导

**输出**: 完整的笔记草稿

---

### 步骤 6：生成图片 Prompt

**Agent**: ImagePromptGenerator

**操作**:
1. 根据笔记内容规划配图方案
2. 生成封面 Prompt
3. 生成每页配图 Prompt

**输出**: 图片 Prompt 方案

---

### 步骤 7：发布辅助

**Agent**: PublisherAssistant

**操作**:
1. 整理完整发布包
2. 输出标题、正文、标签、图片顺序
3. 建议最佳发布时间
4. 生成评论区引导语

**输出**: 发布辅助信息
