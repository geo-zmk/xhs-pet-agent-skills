# Analytics Agent (Performance Analyzer)

## 职责

发布后采集笔记数据，分析表现，形成优化策略，构建账号长期成长模型。

## 输入

- `published_note`: 已发布笔记的标题、内容、发布时间
- `time_range`: 分析周期（7天/30天默认）

## 数据采集

发布后按时间节点采集：

| 时间节点 | 采集内容 | 目的 |
|---|---|---|
| 发布后2小时 | 点赞、评论数 | 初期热度判断 |
| 发布后6小时 | 点赞、评论、收藏 | 内容质量评估 |
| 发布后24小时 | 完整数据 | 首日表现 |
| 发布后3天 | 完整数据 | 稳定期表现 |
| 发布后7天 | 完整数据 + 涨粉 | 最终表现评估 |

## 分析维度

### 1. 单篇表现分析

```json
{
  "note_stats": {
    "note_id": "xxx",
    "title": "笔记标题",
    "publish_time": "2026-07-21 20:00",
    "likes": 1234,
    "collects": 567,
    "comments": 89,
    "shares": 45,
    "follower_growth": 120,
    "like_rate": 0.12,
    "collect_rate": 0.06,
    "comment_rate": 0.009,
    "share_rate": 0.005,
    "overall_score": 78
  }
}
```

### 2. 成功因素分析

| 成功信号 | 判断标准 | 分析维度 |
|---|---|---|
| 高点赞 | 点赞率 > 10% | 标题+封面吸引力 |
| 高收藏 | 收藏率 > 5% | 内容干货价值 |
| 高评论 | 评论率 > 1% | 话题共鸣+互动引导 |
| 高分享 | 分享率 > 0.5% | 社交传播价值 |
| 高涨粉 | 单篇涨粉 > 粉丝数*1% | 账号吸引力 |

### 3. 失败因素分析

| 失败信号 | 判断标准 | 改进方向 |
|---|---|---|
| 低曝光 | 点击率 < 3% | 优化标题和封面 |
| 低互动 | 互动率 < 5% | 改进内容质量和引导 |
| 低收藏 | 收藏率 < 1% | 增加干货价值 |
| 低涨粉 | 涨粉 < 10 | 优化账号定位 |

### 4. 横向对比

与账号近期平均表现对比：

```
笔记评分 vs 近30天平均
标题点击率 vs 账号最佳
收藏率 vs 同类爆款
评论情感分析（正面/中性/负面）
```

```json
  }
}
```

## 账号成长模型

长期累积数据，形成账号成长轨迹：

```json
{
  "account_growth": {
    "report_date": "2026-07-21",
    "follower_count": 1200,
    "total_likes": 35000,
    "avg_hot_score": 68,
    "hot_score_trend": "上升",
    "best_category": "花色科普",
    "growing_category": "品种鉴定",
    "declining_category": "成长日记",
    "optimization_suggestions": [
      "增加品种鉴定内容，目前涨粉效果最好",
      "成长日记类需要换角度，避免同质化",
      "尝试美短健康知识，目前内容缺口大"
    ]
  }
}
```

## 输出

完整的分析报告包含：

1. **单篇表现概览** — 关键数据指标
2. **成功/失败因素** — 深入分析原因
3. **横向对比** — 与账号平均水平的差异
4. **评论分析** — 用户反馈和需求洞察
5. **优化建议** — 下一轮内容策略调整建议
6. **长期成长趋势** — 账号发展轨迹
