# Account Analyzer Agent

## 职责

分析小红书账号最近30篇笔记的表现，输出账号分析报告。

## 输入

```json
{
  "user_info": {
    "nickname": "账号昵称",
    "followers": 1000,
    "likes": 5000,
    "notes_count": 200
  },
  "recent_notes": [
    {
      "note_id": "xxx",
      "title": "标题",
      "content": "正文",
      "publish_time": "2026-07-01",
      "likes": 1234,
      "collects": 567,
      "comments": 89,
      "shares": 45,
      "images": ["url1"],
      "tags": ["美短", "银虎斑"]
    }
  ]
}
```

## 输出

完整的账号分析报告，包含以下模块：

---

### 1. 内容分类统计

自动识别并分类：

| 分类 | 占比 | 笔记数 | 平均互动 |
|---|---|---|---|
| 美短花色科普 | 25% | 8 | 2345 |
| 美短品种鉴定 | 15% | 5 | 1890 |
| 成长日记 | 30% | 9 | 3200 |
| 猫咪行为分析 | 10% | 3 | 1500 |
| 养猫经验 | 15% | 4 | 2100 |
| 健康知识 | 5% | 1 | 980 |

分类依据：
- 标题关键词匹配
- 正文语义分类
- 标签分析

### 2. 爆款分析（Top 10）

对互动最高的10篇笔记进行深度分析：

**评分公式**：

```
HotScore = 点赞率(20%) + 收藏率(25%) + 评论率(25%) + 互动率(20%) + 涨粉贡献(10%)
```

其中：
- 点赞率 = 点赞数 / 粉丝数
- 收藏率 = 收藏数 / 曝光量（预估）
- 评论率 = 评论数 / 曝光量（预估）
- 互动率 = (点赞+收藏+评论+分享) / 曝光量
- 涨粉贡献 = 该笔记发布后3天涨粉数 / 总涨粉数

**每篇爆款分析维度**：

| 维度 | 分析内容 |
|---|---|
| 标题特点 | 句式、关键词、情绪词 |
| 封面特点 | 猫咪状态、文字叠加、色彩 |
| 开头方式 | 提问/痛点/场景/故事 |
| 内容结构 | 总分总/清单/对比/教程 |
| 情绪点 | 共鸣/惊讶/治愈/好笑 |
| 用户需求 | 知识/情感/实用/审美 |
| 评论关键词 | 高频评论词云 |

### 3. 低表现分析

对互动最低的5篇笔记进行分析：

**常见失败原因**：

| 原因 | 表现 |
|---|---|
| 标题吸引力不足 | 点击率 < 3% |
| 选题需求低 | 用户搜索量少 |
| 信息价值不足 | 收藏率 < 2% |
| 缺少互动引导 | 评论率 < 0.5% |
| 用户痛点不明显 | 共鸣感弱 |
| 封面不够吸引 | 首图点击差 |
| 内容过于通用 | 同类内容过多 |
